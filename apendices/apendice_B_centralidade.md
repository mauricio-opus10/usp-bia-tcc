# Apêndice B — Tabelas Completas de Centralidade

As tabelas a seguir apresentam as métricas de centralidade calculadas para cada Estado em cada camada da rede multiplex, nos dois períodos analisados. Os valores foram obtidos com o pacote **NetworkX** (Python) a partir dos dados do **Correlates of War Project**. Os Estados estão ordenados por centralidade de grau (decrescente).

> **Nota:** este apêndice corresponde ao Apêndice B da versão completa do TCC. Foi removido do PDF submetido para cumprir o limite de 30 páginas e disponibilizado neste repositório para consulta da banca.

## Reprodutibilidade

Os valores destas tabelas são gerados pela pipeline em `src/network/metrics.py` e exportados em `data/output/comparison/centralidade_novas.csv` (versão consolidada também disponível em [`apendice_B_centralidade.csv`](apendice_B_centralidade.csv) — 99 linhas: 4 camadas × 2 períodos × ~12 Estados, com colunas `degree`, `betweenness`, `closeness`, `eigenvector`).

Para regerar:

```bash
~/.virtualenvs/datascience/bin/python run_pipeline_comparison.py
```
